Magisk教程

以下操作，为一般手机或类原生Android且换了国内源教程（因为补丁需要下载卡刷包），不同手机、不同Rom可能不一样（如：坚果3）。

卡刷安装

前提：已经安装好Twrp
在已经安装Twrp的情况下，卡刷Magisk的卡刷包，以V19.3为例：

在Twrp下选择对应包后，卡刷、开机。如果开机正常，万事大吉，否则请看卸载内容。

对内核补丁安装

前提：提取了系统内核（boot.img)文件
提取了系统内核boot.img情况下，安装好Magisk Manager App，选择安装->安装方式：选择并修补一个文件，这个时候选择我们的boot.img：

如图，生成一个magisk_boot.img。
然后，Twrp下刷入magisk_boot.img或者电脑fastboot flash boot magisk_boot.img。
如果开机正常，万事大吉，否则请看卸载内容。

直接安装

前提：手机本身有root
在Magisk Manager App里选择直接安装，然后重启手机。

卸载Magisk

卸载方法太多了，这边推荐两个比较靠谱、简单方法：

  1. 有Twrp情况下，刷入原版boot.img/刷入Magisk卸载卡刷包。
  2. 没有Twrp情况下，fastboot flash boot boot.img。

Miui安装Magisk

MIUI安装很简单，Miui开发版自带不完全Root；使用Miui开发版不完全Root授权Magisk Manager App，然后选择直接安装，重启，即可完成安装。
（该方法已经和Aomorile主题开发者青森乐测试通过）
